Ghostscript

Resize PDF to Letter

gs \
 -o output.pdf \
 -sDEVICE=pdfwrite \
 -sPAPERSIZE=letter \
 -dFIXEDMEDIA \
 -dPDFFitPage \
 -dCompatibilityLevel=1.4 \
  input.pdf

Device size parameter '-dPDFSETTINGS'

Ex

gs \
 -o output.pdf \
 -sDEVICE=pdfwrite \
 -sPAPERSIZE=letter \
 -dFIXEDMEDIA \
 -dPDFFitPage \
 -dPDFSETTINGS=/ebook \
 -dCompatibilityLevel=1.4 \
  input.pdf

Combine/Merge PDFs and maintain links

gs -q -dNOPAUSE -dBATCH -sDEVICE=pdfwrite -sOutputFile=merged.pdf source1.pdf source2.pdf source3.pdf etc.pdf

Or

gs -dNOPAUSE -sDEVICE=pdfwrite -sOUTPUTFILE=combine.pdf -dBATCH regex*

Crop Alternating and Combine with pdftk

gs                           \
  -o cropped.pdf             \
  -sDEVICE=pdfwrite          \
  -dDEVICEWIDTHPOINTS=370    \
  -dDEVICEHEIGHTPOINTS=290   \
  -dFIXEDMEDIA               \
  -c "24 72 translate"       \
  -c " 0 0 535 722 rectclip" \
  -f uncropped-input.pdf

To use, replace 3 instances of with the source pdf

gs                               \
  -o cropped1.pdf                 \
  -sDEVICE=pdfwrite              \
  -c "[/CropBox [120 417 490 707]" \
  -c " /PAGES pdfmark"           \
  -f <name>.pdf; gs                               \
  -o cropped2.pdf                 \
  -sDEVICE=pdfwrite              \
  -c "[/CropBox [120 87 490 377]" \
  -c " /PAGES pdfmark"           \
  -f <name>.pdf; pdftk A=cropped1.pdf B=cropped2.pdf shuffle A B output <name>.pdf; rm cropped1.pdf; rm cropped2.pdf
